Introduction

HTML stands for HyperText Markup Language. It is used to create web pages and web applications. HTML describes the structure of a web page using elements and tags.

HTML Logo

History of HTML

HTML was created by Tim Berners-Lee in 1991. It has evolved over time, and the latest version is HTML5, which supports audio, video, and advanced graphics.

Uses of HTML

  • Creating web pages
  • Structuring website content
  • Embedding images and videos
  • Creating forms
  • Linking web pages

Steps to Create a Web Page

  1. Create an HTML file
  2. Write basic HTML structure
  3. Add content using tags
  4. Save the file
  5. Open it in a browser
